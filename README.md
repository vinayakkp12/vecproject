# Vec Project

React-based login and index pages using modern HTML/CSS and React 18 CDN.

## Files

- `login-react.html` - Login page with form & authentication UI
- `index-react.html` - Main index/home page

## Features

- Modern dark theme design
- Responsive layout
- React 18 integration via CDN
- Smooth animations & transitions

## Getting Started

Open either HTML file directly in a browser. No build step required!
